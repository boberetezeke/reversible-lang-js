Examples

Input a name and display it

name = input_string ( "enter name" )
output ( "your name is" )
output ( name )

Get a list of food bills, total and average them

sum = 0
num_people = 0
value = 1
while value != 0
  value = input_number ( "enter food amount" )
  if value != 0
    sum = sum + value
    num_people = num_people + 1
  end
end
output ( "total bill is" )
output ( sum )
output ( "the average bill is" )
output ( sum / num_people )

Syntax

  • Assignment: identifier '=' Expression
  • Expression: (Literal | Identifier) operand (Literal | Identifier)
  • FunctionCall: Identifier '(' Argument* ')'
  • If Statement:
    • 'if' Expression
    • Statements*
    • 'else' (else clause optional)
    • Statements*
    • 'end'
  • While Statement:
    • 'while' Expression
    • Statements*
    • 'end'
  • Literal: \d+ | "[^"]*"
  • Identifier: [\w_]+

Functions

  • Input a number - input_number ( "a string prompt" )
  • Input a string - input_string ( "a string prompt" )
  • Output a value - output ( literal_or_variable_name )
