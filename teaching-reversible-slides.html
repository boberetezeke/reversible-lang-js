What is a computer?

Memory

Instructions

Instructions - Use an English like language

  • It is a cross between mathematical formulas and English
  • Names are given to locations in memory
  • Math expressions are used to create values to put into memory associated with a name
  • English statements are used to make decisions like "if" and "while"

Memory - Store Information

  • Ask the user and then put it in
    • name = input_string ( "what is your name?" )
  • Use an instruction to put it in directly
    • name = "Steve"

Memory - Retrieve Information

  • refer to what is in memory by name and use it in an expression
    • output ( "hello " + name )

Make Decisions

  • Ask if something is true
    • if name == "Steve"
      • output ( "I like that name" )
    • end

Repeat Until Done

  • Do something three times
    • number = 1
    • while number <= 3
      • output ( "this is number: " + number )
    • end

Two Kinds of Equal

  • = - means to store information in memory
  • == - see if two pieces of information are the same

Different Types of Information

What defines a type is what operations can be performed on that piece of information.

  • In this language there are three types of information.
    • Numbers - any number
    • Strings - strings of letters, numbers or symbols
    • Booleans - truth values -- either true or false

Numbers

  • Operations
    • + - Addition
    • - - Subtraction
    • * - Multiplication
    • / - Division
    • % - Remainders

Strings

  • Operations
    • + - Concatenation
      • "hello " + "world" ==> "hello world"
  • You define a string by surrounding it with double quotes

Booleans

  • Operations
    • && - AND two values together
      • tall = height > 72
      • skinny = weight < 100
      • if tall && skinny
      • output ( "you are a bean-pole" )
      • end
    • || - OR two values together
      • if tall || skinny
      • output ( "you may be a bean-pole" )
      • end

Solve A Problems

Ask each person what they paid for their bill.
At the end enter zero for the bill amount.
Display the total bill amount, and the average amount

A Solution

sum = 0
num_people = 0
value = 1
while value != 0
  value = input_number ( "enter food amount" )
  if value != 0
    sum = sum + value
    num_people = num_people + 1
  end
end
output ( "total bill is" )
output ( sum )
output ( "the average bill is" )
output ( sum / num_people )
