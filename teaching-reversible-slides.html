What is a computer?

Memory

Instructions

Instructions - Use an English like language

  • It is a cross between mathmatical formulas and english
  • Names are given to locations in memory
  • Math expressions are used to create values to put into memory associated with a name
  • English statements are used to make decisions like "if" and "while"

Memory - Put stuff in

  • Ask the user and then put it in
    • name = input_string ( "what is your name?" )
  • Use an instruction to put it in directly
    • name = "Steve"

Memory - Get stuff out

  • refer to what is in memory by name and use it in an expression
    • output ( "hello " + name )

Solve A Problems

Ask each person what they paid for their bill.
At the end enter zero for the bill amount.
Display the total bill amount, and the average amount

Minimum Functionality

  • Get Input
  • Write to memory
  • Read from memory
  • Have Types: Number, String, Boolean
  • Put Output
  • Make Decisions
  • Conditionally Loop

Design Decisions

Syntax

  • Space Delimited Everything
  • One Statement Per Line: Assignment | FunctionCall
  • Assignment: identifier '=' Expression
  • Expression: (Literal | Identifier) operand (Literal | Identifier)
  • FunctionCall: Identifier '(' Argument* ')'
  • If Statement:
    • 'if' Expression
    • Statements*
    • 'else' (else clause optional)
    • Statements*
    • 'end'
  • While Statement:
    • 'while' Expression
    • Statements*
    • 'end'
  • Literal: \d+ | "[^"]*"
  • Identifier: [\w_]+

A Solution

sum = 0
num_people = 0
value = 1
while value != 0
  value = input_number ( "enter food amount" )
  if value != 0
    sum = sum + value
    num_people = num_people + 1
  end
end
output ( "total bill is" )
output ( sum )
output ( "the average bill is" )
output ( sum / num_people )
